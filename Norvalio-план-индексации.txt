ЧАСТЬ C. ЯНДЕКС.ВЕБМАСТЕР
-------------------------
Ресурс: добавить https://norvalio.com  (ОБЯЗАТЕЛЬНО как https, НЕ http — иначе данные раздвоятся).

Если сайт ещё НЕ добавлен:
  1. webmaster.yandex.ru → «Добавить сайт» → https://norvalio.com
  2. Подтвердить прав — HTML-файлом в корень (положить файл, залить на GitHub, НЕ удалять) или метатегом/DNS.
  3. «Индексирование → Файлы Sitemap» → добавить sitemap.xml (и/или sitemap-guides.xml).
  4. «Информация о сайте → Регион сайта» → задать регион: Дубай (ОАЭ).

Ускорить обход новых страниц:
  «Индексирование → Переобход страниц» → вставлять по одному полному адресу из списка выше → отправить.
  (Лимит переобхода в Яндексе небольшой — по несколько URL в день; остальное Яндекс возьмёт из sitemap сам.)

Через 5-7 дней: «Индексирование → Страницы в поиске» — проверить, что статьи попали.
