ЧТО В ПАКЕТЕ
  agents.html                                  — панель управления (Команда / Лента / Доска / Статьи / Настройки)
  netlify.toml                                 — настройки Netlify (НОВЫЙ файл в корне)
  package.json                                 — зависимости (НОВЫЙ файл в корне)
  netlify/functions/seo-agent.mjs              — API панели
  netlify/functions/seo-agent-run-background.mjs — мозг: цепочка из 6 агентов
  netlify/functions/seo-agent-cron.mjs         — автозапуск каждое утро 07:00 (Дубай)

────────────────────────────────────────────────
ШАГ 1. ЗАГРУЗИТЬ ФАЙЛЫ В GITHUB (репо aapolevikov/norvalio)
────────────────────────────────────────────────
1. Откройте github.com/aapolevikov/norvalio
2. Add file → Upload files
3. Перетащите ВСЮ распакованную структуру:
   - agents.html, netlify.toml, package.json — в корень
   - папку netlify (с подпапкой functions и 3 файлами внутри)
   ВАЖНО: при перетаскивании папки netlify целиком GitHub сам сохранит структуру.
4. Commit changes
5. Netlify задеплоит автоматически (1–3 минуты)

────────────────────────────────────────────────
ШАГ 2. КЛЮЧ CLAUDE API
────────────────────────────────────────────────
1. console.anthropic.com → Sign up (вход через Google)
2. Меню слева: Billing → добавить карту (минимальное пополнение $5 хватит на ~1.5 месяца работы команды)
3. Меню слева: API Keys → Create Key → имя "norvalio-seo" → Copy
   (ключ показывается ОДИН раз — сразу копируйте)

────────────────────────────────────────────────
ШАГ 3. ВСТАВИТЬ КЛЮЧ В NETLIFY
────────────────────────────────────────────────
1. app.netlify.com → войти через GitHub → выбрать сайт norvalio
2. Site configuration → Environment variables → Add a variable
3. Key:   ANTHROPIC_API_KEY
   Value: (вставить ключ из шага 2)
4. Save → Deploys → Trigger deploy → Deploy site

(Необязательно, защита панели: добавьте вторую переменную PANEL_KEY
 с любым паролем — тогда панель попросит его в Настройках.)

────────────────────────────────────────────────
ШАГ 4. ПРОВЕРКА
────────────────────────────────────────────────
1. Откройте https://norvalio.com/agents.html
2. Статус вверху должен стать «Бэкенд подключён» (зелёная точка)
3. Нажмите «▶ Запустить смену»
4. Перейдите во вкладку «Лента» — увидите, как агенты передают
   работу друг другу (Стратег → Автор → Редактор → SEO-техник →
   Контролёр → Секретарь). Смена занимает 3–6 минут.
5. Готовая статья появится во вкладке «Статьи»:
   «Скачать HTML» → загрузить файл в GitHub → добавить URL
   в sitemap-guides.xml → запросить индексацию в GSC.

────────────────────────────────────────────────
КАК ЭТО РАБОТАЕТ ДАЛЬШЕ
────────────────────────────────────────────────
• Каждое утро в 07:00 команда сама пишет статью (cron).
• Задачи «Ждёт Александра» на Доске — это то, что без вас
  не решить (залить файл, запросить индексацию).
• Контент-план (30 тем) редактируется в Настройках.
• Фаза 2 (по готовности): Аналитик с GSC API, Публикатор
  с GitHub-токеном (сам заливает статьи), агенты для Mister Ride.

СТОИМОСТЬ: ~$0.10–0.15 за смену ≈ $3–5/месяц при ежедневной работе.
